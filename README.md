# Youtube-mini

## global router
/ -> home (trending video)
/join -> join (user)
/login -> login (user)
/search -> search

## user router
/users/:id -> see user
/users/logout -> log out
/users/edit -> edit my profile
/users/:id/remove -> remove my profile

## video router
/videos/:id -> see video
/videos/:id/edit -> edit video
/videos/:id/delete -> delete video
/videos/upload -> upload video
